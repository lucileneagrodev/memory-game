Projeto Memory-Game-JS-Dio

Este é um projeto de jogo de memória com emojis utilizando javaScript, desenvolvido
como parte de um desafio da Digital Innovation One (DIO). É um jogo interativo com 
figuras de emojis animados e cores alegres, jogo muito usado para exercitar a nossa memória.


Funcionalidades

- Quadrados com imagens de emojis:Exibe quadrados.
- Detalhes:Quando clica em um quadrado a carta vira e aparece a imagem de um emoji, logo depois 
deve ser clicado em outro quadrado e as imagens dos dois emojis devem ser iguais, se as imagens não se 
forem iguais, o jogo volta os quadrados das figuras na pocisão original.E o jogador pode tentar de novo,
isso permite a pessoa treinar a memória para gravar a posição das figuras.Se o jogador acertar todas as cartas 
aparece uma mensagem, informando que a pessoa ganhou, nesse caso existe um botão para resetar 
o jogo e voltar ao inicio, ai as figuras mudarão de posição para a proxima jogada.

Tecnologias Utilizadas

-HTML5 e CSS3
-JavaScript
